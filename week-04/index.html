Part 1

Box Model — Margin, Padding, Border

Box One — Small Padding (12px)

This box has padding: 12px — notice how the text sits close to the border. The border is the solid blue line; the margin below creates the gap before the next box.

Box Two — Medium Padding (28px)

With padding: 28px the content is pushed further inward. The border still wraps the content and the padding. The same margin separates it from the boxes above and below.

Box Three — Large Padding (44px)

padding: 44px creates a noticeably spacious interior. The outer box is larger even though the content itself is the same size — that's the box model at work: the border grows to accommodate the padding.

Positioning — Relative & Absolute

NEW

This container uses position: relative. It stays in the normal document flow, but it now acts as a coordinate reference point for any absolutely-positioned children.

The red "NEW" badge uses position: absolute with top: 12px; right: 12px;. It is removed from normal flow and placed precisely at the top-right corner of this container — not of the page.

Part 3

Float & Clear — Flow Manipulation

Main Content (float: left)

This div uses float: left. It is removed from the normal vertical stacking flow and pushed to the left edge of its container. The sidebar fills the space to the right.

Floats were originally designed for text wrapping around images, but were widely adopted for multi-column layouts before Flexbox and Grid became available.

Sidebar (float: right)

float: right pushes this element to the right edge. Both floated elements sit side by side on the same line.
